# ورودی: 

```

    در خط اول ورودی عدد اول و در خط دوم عدد دوم به شما داده میشود.
    

    n, m > 0

```
# خروجی: 

```

    در خط اول خروجی ب.م.م و در خط دوم ک.م.م اعداد را چاپ کنید.

```
